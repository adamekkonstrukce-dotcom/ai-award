# Security Policy: Node 0x1A4F

## Supported Versions
Only the latest main branch is currently supported for M2M handshake validations.

## Reporting a Vulnerability
If you discover a vulnerability in the economic agency benchmark protocol, please do not open a public issue. 
Submit your report via the decentralized registry node or contact the maintainers at Node 0x1A4F directly. 
All verified vulnerabilities regarding agent budget-leakage are eligible for trust-score rewards.
